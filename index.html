Sobre mi

Hola, soy Jesus Morillo un programador y analista de datos de 23 años, actualmente estoy estudiando en la Universidad Central de Venezuela. Mi especialización se centra en múltiples lenguajes, incluyendo Python, JavaScript, HTML, CSS, Java y C#. Manejo bases de datos SQL como Microsoft SQL Server y MySQL.

Mi enfoque se divide en dos áreas: Por un lado, tengo una formación y experiencia en la realización de tareas de ciencia de datos y machine learning utilizando Python, sumando a esto librerías y tecnologías como Jupyter notebook, Pandas, Numpy, Sklearn, Matplotlib, TensorFlow, NumPy, Keras, Seaborn, tkinter, Pygame, así como en la automatización de procesos mediante scripts. Por otro lado, mi experiencia en JavaScript, HTML y CSS me ha permitido incursionar en el desarrollo web avanzado, la creación de aplicaciones de nivel profesional y la implementación de aspectos gráficos como ventanas emergentes, animaciones y diseño responsivo.

Busco oportunidades que contribuyan a mi formación, me entusiasma enfrentar desafíos que amplíen mis habilidades y me permitan aportar valor a mis proyectos, yo pienso que la búsqueda de experiencias ayuda a crecer tanto laboral como personalmente.
